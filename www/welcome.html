Stellarium for Java is a Java version of Stellarium, a free GPL software which renders realistic skies in real time. With Stellarium, you really see what you can see with your eyes, binoculars or a small telescope. Stellarium is also used in planetariums.

Stellarium for Java aims to provide the same functionnalities as the original Stellarium but over the Java platform, thus allowing both a greater portability accross platforms and low level APIs (Java3D, Direct3X) and evolutivity, as well as ligther and easer coding.

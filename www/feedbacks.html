Feedback

Support, bug reports, feature requests and feedbacks

As this project is free, we need your help and motivation to improve it and fix the bugs, so please feel free to send reports, or just tell how you like/hate the software! If you feel like contributing in stellarium, please have a look at the development page.

Sourceforge.net provides a good way of submitting/reading feedbacks, bug reports, features requests or patches.

  • Support
    If you need technical assistance (e.g. installation problems)
  • Bug reports
    You find a bug? It's here (please first check if the bug is already reported)
  • Feedbacks
    Public forum for thank/hate letters or other non technical things
  • Feature Requests
    For things that you would like to see in Stellarium
  • Patches
    Here you can find released patches to fix some bugs. And of course, if you are a programer, we will be very happy if you submit your own patch

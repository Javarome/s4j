Requirements

  • J2SE Runtime Environment (JRE) 5.0 or higher.
  • Java 3D 1.3 or higher.
  • JOGL 1.1.1 or higher
  • A 3D Card with a support for OpenGL or Direct3X. At least a Voodoo3 or a TNT2 is recommended for smooth animation.
  • A dark room for realistic rendering : details like the Milky Way or star twinkling can't be seen in a bright room.

SVN version

You can browse the SVN trunk tree from here.
To get the latest SVN snapshot (from sourceforge) just checkout (using Tortoise SVN) or (using Tigris SubVersion command line tools) type in a console:

svn co https://svn.sourceforge.net/svnroot/stellarium4java stellarium4java

then to build from the sources, type:

cd stellarium4java
ant

finally try a local run:

cd classes
java Main
